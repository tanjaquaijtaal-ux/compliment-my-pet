### 1. Banner Image (`banner.png`)
- **What**: A welcoming hero image for the README
- **Suggested content**: Your agent's icon/logo or a sample interaction
- **Dimensions**: Recommended 1200x400px or similar wide format
- **Format**: PNG or JPG

### 2. Main Interface (`main-interface.png`)
- **What**: The agent's chat interface in Copilot Studio or when deployed

- **Format**: PNG preferred

### 3. Example Interaction (`example-interaction.png`)
- **What**: A real conversation showing the agent complimenting a pet
- **Show**: User message with pet photo and agent's enthusiastic response
- **Include**: Full conversation flow demonstrating the agent's personality
- **Dimensions**: Capture the full conversation window
- **Format**: PNG preferred

### 4. Agent Configuration (`agent-configuration.png`)
- **What**: The agent setup/configuration view in Copilot Studio
- **Show**: Topics, entities, or the agent overview page
- **Note**: Make sure no sensitive information is visible
- **Dimensions**: Standard screenshot
- **Format**: PNG preferred

## üìù How to Take Screenshots

### From Copilot Studio:
1. Navigate to the view you want to capture
2. Use your system's screenshot tool:
   - **Windows**: Win + Shift + S
   - **Mac**: Cmd + Shift + 4
   - **Linux**: Use your distribution's screenshot tool
3. Save with the appropriate filename
4. Place in this `screenshots` folder

### From Deployed Agent:
1. Open the agent in its deployed environment (Teams, website, etc.)
2. Interact with it using a pet photo
3. Capture the conversation
4. Save and add to this folder

## ‚ú® Screenshot Best Practices

- **Resolution**: Use high-quality screenshots (at least 1080p)
- **Format**: PNG for UI screenshots (better quality), JPG for photos
- **Privacy**: Blur or remove any personal information
- **Lighting**: Ensure good visibility and contrast
- **Cropping**: Remove unnecessary browser chrome or toolbars
- **File size**: Optimize images to keep repository size reasonable
  - Use tools like [TinyPNG](https://tinypng.com/) or [ImageOptim](https://imageoptim.com/)
